## 6. 功能依赖关系

```
配方管理 ──┬── 可视化编辑 ──┬── 智能布局
           │               │
           │               └── 工艺段管理
           │                      │
           │                      └── 子步骤管理
           │
           ├── 实时协作 ──── 动态字段配置
           │
           └── 设备调度
                    │
                    └── 配置管理
```

**依赖说明**:
- 配方管理是最基础模块，其他模块都依赖它
- 可视化编辑依赖工艺段管理和智能布局
- 智能布局依赖工艺段管理的数据
- 实时协作依赖配方管理的实时同步
- 设备调度依赖工艺段和子步骤数据
- 配置管理为其他模块提供配置支持
